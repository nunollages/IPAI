# Project Overview

The goal of this project is to design and implement a **domain-specific search engine** supported by a **data warehouse**. The system integrates data from multiple sources and enables efficient retrieval and ranking of information.

This work combines three key areas:

- **Data Integration**
- **Data Warehousing**
- **Information Retrieval**
